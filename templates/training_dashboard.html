3. Store Category Entry

Store category is used to group store items.

Category Name: Copper Pipe

4. Store Item Entry

Store item is actual material available in store.

Category: Copper Pipe

Item Description: Copper Pipe

Size: 1/4 Soft Pipe

Unit: MTR and RFT

Opening Stock: 500

Minimum Stock: 100

Remarks: Used for HVAC indoor to outdoor connection

5. Store Stock In Entry

Stock In is used when new material is purchased and added to store.

Item: Copper Pipe - 1/4 Soft Pipe

Transaction Type: Stock In

Purpose: Purchase

Quantity: 200

Description: Purchased new copper pipe stock

6. Store Stock Out Entry

Stock Out is used when material is issued from store.

Item: Copper Pipe - 1/4 Soft Pipe

Transaction Type: Stock Out

Purpose: Project Work

Project: Mohali Warehouse HVAC Installation

Quantity: 50

Issued To: Munish Kumar

Description: Material issued for site installation

7. BOQ Entry

BOQ means Bill of Quantity. It stores required project material.

Project: Mohali Warehouse HVAC Installation

BOQ Title: HVAC Installation BOQ

Status: Draft

Remarks: Material planning for first phase installation

8. BOQ Item Entry

BOQ item defines how much material is required for the project.

Store Item: Copper Pipe - 1/4 Soft Pipe

Required Quantity: 200

Rate: 250

Total Amount: 50000

Remarks: Required for AC piping work

9. Material Issue Entry

Material Issue records material sent from store to project site.

Project: Mohali Warehouse HVAC Installation

BOQ: HVAC Installation BOQ

Issued To: Munish Kumar

Received By: Site Engineer

Status: Issued

Remarks: First material issue for installation

10. Material Issue Item Entry

Material issue item records the exact quantity issued to site.

Issued Quantity: 50

Consumed Quantity: 40

Returned Quantity: 5

Balance Quantity: 5

Remarks: 40 used, 5 returned, 5 pending at site
